# carefree-ml

`carefree-ml` implemented Machine Learning algorithms with numpy, mainly for educational use.


## Installation

`carefree-ml` requires Python 3.6 or higher.

```bash
git clone https://github.com/carefree0910/carefree-ml.git
cd carefree-ml
pip install -e .
```
